## News

Jan/2022: Added PWA technology


## Tools used
- React
- Redux
- Octicons
- Parcel Bundler
- Local Storage

## Commands

Run development server (port 8080):
```
  npm start
```

Run build (default folder 'dist'):
```
  npm build
```
